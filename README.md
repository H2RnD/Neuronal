# Neuronal
Neuronal is an experimental framework for building and evolving biologically inspired neural networks that learn reasoning tasks with adaptive modality.

Key Characteristics
	-Extremely lightweight and interpretable networks
	-Evolves over generations using feedback-driven mutations, lineage tracking, and protective mechanisms to prevent catastrophic forgetting 
	-Designed to explore how small networks can discover and specialize in rule-like behaviors (arithmetic, language, other media I/O) without relying on massive scale 
	-Supports controlled ablation studies of routing, gating, composition, and protection strategies 

Core Goals
	-Understand the minimal conditions under which a tiny network can learn genuine cross-modal symbolic generalization. (Such as mathematically “perfect” neural nodes, that can be engineered and duplicated)
	-Probe the boundaries of representation, routing, and protection mechanisms in spiking-like architectures 
	-Identify architectural patterns that allow precise, rule-like computation in very low-capacity systems 

Current Focus Areas
	-Closed-domain arithmetic (small integers, addition/subtraction)
	-Unified math + language pipelines 
	-Order-preserving input composition 
	-Modality-aware routing and integration 
	-Long-run convergence behavior in symbolic domains
